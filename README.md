# AI Zukko Bot

AI Zukko Bot is a Telegram-based AI assistant bot.  
It supports chat responses using an AI API (e.g., Gemini/OpenAI) and stores user conversation history in SQLite.

## Features
- Telegram bot commands and chat handling
- AI-generated responses (API integration)
- SQLite database for chat history
- Optional image handling (PIL)

## Tech Stack
- Python
- pyTelegramBotAPI (telebot)
- SQLite
- Pillow (PIL)
- AI API SDK (Gemini/OpenAI)
